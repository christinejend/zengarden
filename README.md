Voici les directives de soumission pour le nouveau et amélioré csszengarden.com:

- CSS3? Bien sûr! Préfixe pour tous les navigateurs, si nécessaire.
- Aller sensibles; testez votre aménagement à plusieurs tailles d'écran.
- Votre navigateur tests de base: + IE9, Chrome récente / Firefox / Safari et iOS / Android
- La dégradation gracieuse est acceptable, et en fait fortement encouragée.
- Classes d'utilisation pour le coiffage. Ne pas utiliser les identifiants.
- Polices Web sont cool, assurez-vous juste que vous avez un permis de partager les fichiers. Hébergé
les services qui sont appliquées par l'intermédiaire du fichier CSS (ie. Google Fonts) fonctionnera bien, mais
qui nécessitent plus HTML personnalisé ne sera pas. TypeKit est pris en charge, consultez le fichier readme sur cette
page pour les instructions d'utilisation: https://github.com/mezzoblue/csszengarden.com/

Et quelques conseils sur la construction de votre fichier CSS:

- Utilisation: premier enfant,: last-child et: nth-child pour obtenir à des éléments non-classés
- Utilisation :: :: avant et après pour créer des pseudo-éléments pour le coiffage supplémentaire
- Utiliser plusieurs images de fond à appliquer autant que vous avez besoin de tout élément
- Utiliser la méthode Kellum pour le remplacement d'image, si encore nécessaires. http://goo.gl/GXxdI
- Ne pas compter sur les divs supplémentaires au fond. Utilisez :: :: avant et après la place.
